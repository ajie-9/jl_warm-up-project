# 热身 App 网站项目

这是根据 `warmup-app-mvp-prd` 落地的可运行网站项目，核心体验包括：

- 每日早晨食谱提醒
- 个性化热身设置
- 热身方向选择
- 热身前自检与不适提示
- 动态热身跟练
- 我的主页、最近记录和训练后饮食计划

## 技术方案

- Vite：轻量前端工程和本地开发服务器。
- React：承载单页 App 的页面切换、表单选择和训练状态。
- Lucide React：统一图标风格，减少手写 SVG。
- CSS Modules 风格的独立样式文件：便于继续按 PRD 调整视觉细节。

## PRD 覆盖

- App 开屏后的首页与今日热身方向选择。
- 性别、BMI / 身高体重、运动经历、饮食计划意愿等个性化设置。
- 训练前自检、不适提示、正常进入动态热身。
- 胸、背、肩、腿、全身五类热身流程。
- 训练结束后的我的主页、数据记录和训练后饮食入口。

## 运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
